\# Entrega Sesión 1 - Flask API



\## Requisitos

\- Python 3.11

\- Flask



\## Instrucciones

1\. Ejecutar el servidor: `python app.py`

2\. Importar la colección `requests\_session1.json` en Postman.

3\. Probar los endpoints de Tareas y Usuarios.



\## Funcionalidades implementadas:

\- CRUD de Tareas con validación de contenido vacío.

\- CRUD de Usuarios con estructura de dirección anidada.

\- Manejo de IDs únicos mediante contadores globales.
